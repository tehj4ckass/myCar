## 2024-06-20 - German Localization Standard for Dashboard UI
**Learning:** The dashboard UI is primarily localized in German, which applies to all components including micro-interactions like tooltips.
**Action:** When adding or updating user-facing text (like tooltips on buttons), always ensure the language matches the established locale (German), e.g., using 'Daten aktualisieren' instead of 'Refresh Data'.
